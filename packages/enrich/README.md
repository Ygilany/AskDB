# @askdb/enrich

Shared Schema v2 enrichment workspace helpers used by AskDB authoring surfaces.

This package owns the non-UI enrichment model: loading and saving Schema v2
workspaces, building editable table drafts, preserving markdown sections, and
building AI suggestion targets and context.
