# `@askdb/ai-openai`

OpenAI provider adapter for `@askdb/ai`.

```ts
import { createAiRegistry } from "@askdb/ai";
import { openaiProvider } from "@askdb/ai-openai";

const ai = createAiRegistry([openaiProvider]);
```
