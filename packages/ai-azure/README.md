# `@askdb/ai-azure`

Azure OpenAI / Microsoft Foundry provider adapter for `@askdb/ai`.

```ts
import { createAiRegistry } from "@askdb/ai";
import { azureProvider } from "@askdb/ai-azure";

const ai = createAiRegistry([azureProvider]);
```
