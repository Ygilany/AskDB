# `@askdb/ai-google`

Google Generative AI / Gemini provider adapter for `@askdb/ai`.

```ts
import { createAiRegistry } from "@askdb/ai";
import { googleProvider } from "@askdb/ai-google";

const ai = createAiRegistry([googleProvider]);
```
