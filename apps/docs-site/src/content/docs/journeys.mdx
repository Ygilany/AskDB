<p className="doc-eyebrow">Use-case first</p>

<p className="doc-lede">
  Choose the path that matches what you are trying to build. AskDB is split into packages, but most users should not start with package names. Start with the outcome: embed natural-language SQL, author schema context, add retrieval for large schemas, expose an HTTP API, or prepare for agent workflows.
</p>
