<p className="doc-eyebrow">Use-case first</p>

<p className="doc-lede">
  AskDB optimizes for <strong>trust-first analytics</strong> (answers grounded in your schema and explicit boundaries when data touches a model) and a <strong>developer-first embed</strong> (BYO keys, database connectivity, embeddings, and vector stores). Choose the journey that matches what you are shipping—not the package name you might install first.
</p>

<div className="link-cards">
  <a href="../guides/embed-in-app/">

  <a href="../guides/author-enrich-schema/">

  <a href="../guides/rag-large-schema/">
    <strong>Add RAG for large schemas</strong>
    <span>Chunk and index the same Schema v2 artifact with BYO embedder and vector store; retrieve context per question.</span>
  </a>
  <a href="../guides/http-api/">
