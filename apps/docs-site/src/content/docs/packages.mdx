## Monorepo development

<p>If you work inside the AskDB source repository, reusable libraries live under <code>{"packages/*"}</code> and first-party apps under <code>{"apps/*"}</code>. UI surfaces depend on shared headless packages instead of each other. Common workspace commands:</p>



```
pnpm build
pnpm test
pnpm lint
pnpm smoke:install
pnpm preflight
```
