# Support

AskDB is pre-1.0. Public APIs, package boundaries, and CLI behavior may change before 1.0, with changes documented through changesets and package READMEs.

## Questions and Bugs

Use GitHub issues for:

- Installation problems.
- Package or CLI bugs.
- Documentation gaps.
- Feature requests.
- Reproducible SQL generation or validation issues.

Include package versions, Node and pnpm versions, operating system, command output, and a minimal schema fixture when possible. Remove secrets, API keys, customer data, and production database details before posting.

## Security

Report security issues through the process in `SECURITY.md`. Do not include exploit details or private data in public issues.
